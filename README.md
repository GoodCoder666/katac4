# katac4

AlphaZero engine for saiblo Connect4 &amp; Pure Python implementation of some KataGo methods

## Project Structure

```
katac4/
├── README.md        # This file
├── runs/            # TensorBoard logs
├── weights/         # Model checkpoints for each run
│
├── saiblo/          # Saiblo Connect4 submission
│   ├── main.py      # Entry point
│   ├── game.py      # Game environment
│   └── search.py    # Monte-Carlo Graph Search for inference
│
├── train.py         # Main training script
├── model.py         # Model definition (b3c128nbt)
├── game.py          # Game environment (saiblo/game.py without zobrist hash)
├── mcts.py          # Monte-Carlo Tree Search for training
├── elo_eval.py      # Evaluates ELO of model checkpoints
├── elo.json         # Ratings generated by elo_eval.py
├── elo_plot.py      # Plots ELO over time
├── export_model.py  # Exports model to TorchScript
├── benchmark.py     # Model benchmarking
├── human_play.py    # Simple script for human play
```
